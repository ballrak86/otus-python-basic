# Домашнее задание - 2

## Сделать телефонный справочник с использованием классов

## Задание для выполнения
- [x] разбитие кода на модули
      
1. main.py - основной файл в котором запукаем программу, и вызываем controller.py
2. controller.py - вся логика работы справочника, и подключение модулей model.py и view.py
3. model.py - все классы
4. view.py - ввывод для пользователя и обработка набора от пользователя
      
- [x] создание классов
Созданы классы:
1. Contact - инициализация контакта
2. Directory - справочник, наследник Contact. Добавляем id записи и создание контакта через классовый метод, изменение контакат, его поиск, сравнение объектов класса (нужно чтобы понять были ли изменения).
3. FileManager - класс работы с файлами.

- [x] создание исключений

EmptyData и CheckIsDigit - классы исключений, для проверки что данные не пустые, и что данные состоят только из цифр.
