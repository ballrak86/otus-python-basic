# Домашнее задание - 2

## Сделать телефонный справочник с использованием классов

## Задание для выполнения
- [x] разбитие кода на модули
      
1. main.py - основной файл в котором запукаем программу, и вызываем controller.py
2. controller.py - вся логика работы справочника, и подключение модулей model.py и view.py
3. model.py - все классы
4. view.py - ввывод для пользователя и обработка набора от пользователя
      
- [x] создание классов
Созданы классы:
1. Contact - инициализация контакта
2. Directory - справочник, наследник Contact. Добавляем id записи и создание контакта через классовый метод, изменение контакат, его поиск, сравнение объектов класса (нужно чтобы понять были ли изменения).
3. FileManager - класс работы с файлами.

- [x] создание исключений

EmptyData и CheckIsDigit - классы исключений, для проверки что данные не пустые, и что данные состоят только из цифр.

## Запуск программы
Запускаем файл main.py

## Работа с программой
1. Выбираем файл с контактами. По умолчанию можно выбрать уже созданный файл phone_book.json
2. Далее идет работа с меню справочника
```
**************
    Select Phone Book menu:
        1. Save Phone Book
        2. Show all contacts
        3. Create a contact
        4. Find a contact
        5. Edit a contact
        6. Delete a contact
        7. Exit
```
Разберем пункты меню более подробно
1. Сохнанение телефонного справочника
   Первым проходит проверка, нужно ли сохранять справочник. Если не нужно, то сообщаем об этом.
2. Отобразить все контакты.
3. Создание контакта.
   Нужно ввести будет имя, телефон, и комментарий. Тут проходят проверки что нет пустых данных, и что в поле телефон, введены цифры.
4. Поиск контакта.
   Поиск осуществляется по всем параметрам в справочнике. Проверяем что введенные данные не пустые, и так как id и номер телефона хранятся в справочнике с типом int, пытаемся преобразовать введенные данные в int если получится. Если нет, то продолжаем поиск как есть. Поиск ведется по полному совпадению.
5. Изменение контакта.
   Изменение осуществляется по id контакта. Вводим данные имени, телефона и комментария. Проверки на пустые данные и что телефон состоит из цифр.
6. Удаление контакта.
   Удаление по id контакта.
7. Выход из программы.
   Предварительно проверяем были ли изменения в справочнике. Если были то предлагаем сохранить изменения в файл.
